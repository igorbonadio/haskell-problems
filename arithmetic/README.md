# List Problems

## H99-31

Determine whether a given integer number is prime.

_Level_: 1

_Example_:

```haskell
isPrime 7 -- True
```

## H99-32

Determine the greatest common divisor of two positive integer numbers.

_Level_: 1

_Example_:

```haskell
myGCD 36 63     -- 9
myGCD (-3) (-6) -- 3
myGCD (-3) 6    -- 3
```
