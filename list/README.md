# List Problems

## H99-1

Find the last element of a list.

_Level_: 1

_Example_:

```haskell
myLast [1,2,3,4]     -- 4

myLast ['x','y','z'] -- 'z'
```

## H99-2

Find the last but one element of a list.

_Level_: 1

_Example_:

```haskell
myButLast [1,2,3,4]     -- 3

myButLast ['x','y','z'] -- 'y'
```

## H99-3

Find the K'th element of a list. The first element in the list is number 1.

_Level_: 1

_Example_:

```haskell
elementAt [1,2,3] 2   -- 2

elementAt "haskell" 5 -- 'e'
```
